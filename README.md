# binbou jyuku resource planning

小規模個人塾向けの、ローカル中心・低コスト運用を前提とした統合管理システムです。  
本リポジトリは、**【第11期】日本ITチャレンジプロジェクト**の開発課題として進めるプロジェクトです。

- [日本語ドキュメント](./docs/README.ja.md)

## 概要

`binbou jyuku resource planning` は、設備・予算が限られた小規模塾でも運用しやすいように設計した管理システムです。  
高機能な汎用ERPではなく、現場で本当に必要な機能に絞って構成することを前提としています。

## 目的

- 小規模塾の運営を、低コストで効率化する
- 限られた条件の中で、必要十分な機能を設計・実装する
- 課題設定から実装までを通して、問題解決力を高める

## システム方針

- 最小機能優先
- ローカル中心の運用
- Master App + Client App 構成
- 変更履歴を追跡可能な設計

## 想定技術

- **Client**: React Native
- **Server**: Go
- **Database**: SQLite
